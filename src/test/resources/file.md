# MDTool
A tool which can transfer markdown to HTML.

## How it works?
```
MDTool.markdown2Html(new File(markdown_file_path));
```
or 
```
MDTool.markdown2Html(markdown_content);
```

## Why use it？
1. Easy to use
2. Support basic markdwon syntax
3. Support table syntax
